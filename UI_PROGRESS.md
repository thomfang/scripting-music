## 🎨 设计规范（已确立）

### 图标使用

**收藏状态：**
- `heart` (空心，secondaryLabel) - 未收藏
- `heart.fill` (实心，systemPink) - 已收藏

**下载状态：**
- `arrow.down.circle` (accentColor) - 未下载
- `arrow.down.circle.fill` + 进度 (accentColor) - 下载中
- `checkmark.circle.fill` (systemGreen) - 已下载

**播放状态：**
- `waveform` (accentColor) - 正在播放

### 尺寸规范

- **封面：**
  - 搜索页面：56x56
  - Library 页面：48x48
  
- **按钮：**
  - 可点击区域：44x44 或 40x40
  - 图标大小：title3 (约 20pt)

- **间距：**
  - 行间距：12pt
  - 按钮间距：8-12pt

### 颜色规范

- **收藏：** systemPink (已收藏) / secondaryLabel (未收藏)
- **下载：** accentColor (操作中) / systemGreen (已完成)
- **播放：** accentColor
- **文本：** primary / secondaryLabel / tertiaryLabel

## 📝 注意事项

### 已知问题和解决方案

1. **downloadManager.downloadMusic 需要 provider**
   - 解决：使用 `music.provider || "livepoo"` 作为默认值
   - 原因：修复后的代码已保存 provider 信息

2. **下载状态追踪**
   - 使用 `downloadingIds` 数组追踪正在下载的歌曲
   - 下载完成或失败后从数组中移除

3. **封面下载**
   - 收藏时自动在后台下载封面
   - 不阻塞用户操作
